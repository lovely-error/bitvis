This program can produce bitfield layout from given description


```
<input> := <item> (';' <item>)*
<item> := <opts> '|' <seg> (, <seg>)*
<opts> := ('>' | '<')
<seg> := <bitwidth> (<ascii-text> | '#'<bitpattern>)
<bitpattern> := ('1' | '0')+
```

This input...
```
< | 2 IT0, 2 DT, 3 MEMSRC, 6 DSTREG, 3 #000; > | 2 #1, 6 #111000, 8
```
..gives this output
